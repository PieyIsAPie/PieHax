# PieHax
Small and modular WiiU CFW based off of Tiramisu and Aroma

# Included

Region Free Patching (DRC and normal)

WUMS Loader

WiiU Plugin Backend

# How to use
Copy the PieHax environment into the "sd:/wiiu/environments" folder
